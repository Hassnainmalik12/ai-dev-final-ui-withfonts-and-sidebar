## BOOK STORE REACT UI

##### Launch SASS Watcher
```bash
sass src/sass:src --watch --style compressed
```

The above command start SASS Compiler on WATCH Mode.

##### LAUNCH TAILWIND Watcher

```bash
npx tailwindcss -i ./src/main.css -o ./src/index.css --watch --minify
```

##### Launch UI

```bash
yarn start
```
